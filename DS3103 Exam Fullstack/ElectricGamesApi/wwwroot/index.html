ElectricGames API

This is index page. Here you will get a quick introduction on how to use the API.


Endpoints (with links):

  • http://localhost:5137 <-- you are here
  • http://localhost:5137/game
  • http://localhost:5137/game/103
  • http://localhost:5137/game/GetByTitle/Minecraft

How to use:

  1. You navigate though the APIs endpoints using the address bar in your browser.
  2. The domain of the API is http://localhost:5137 .
  3. If you wish to view all records stored in API, enter the domain followed with /game ( you can test this out by clicking the respective link above ).
  4. The different endpoints allow you to filter the results and access different records based on the url entered in the address bar.
  5. Just like the /game shows all records, you can filter the records to a specific id with /game/27 (if the id is 27).
  6. Likewise you can fetch all records that contain a string by using /game/GetByTitle/Minecraft (if the Title is Minecraft).
  7. The browser will auto detect whitespace (spaces) in the title and replace them with '&20' meaning that both /game/GetByTitle/Call&20of&20Duty and /game/GetByTitle/Call of Duty will return the same records.

Enjoy your stay, and keep gaming!
